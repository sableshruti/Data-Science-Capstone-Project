📊 NHANES 2020 Anthropometric Data Analysis

Data Analysis | Statistical Modeling | Exploratory Data Analysis (EDA)

🔎 Project Summary

This project performs an in-depth exploratory data analysis (EDA) on anthropometric measurements from the NHANES 2020 dataset. The objective is to uncover statistical patterns, gender-based differences, and correlations between key body metrics using Python-based data science tools.

The analysis demonstrates practical application of statistical reasoning, data preprocessing, and visualization techniques commonly used in real-world data analytics workflows.

🎯 Business / Analytical Objective

Analyze height, weight, BMI, and waist circumference distributions

Compare statistical differences between male and female populations

Identify correlations between anthropometric variables

Generate derived ratios to enhance insight extraction

Communicate findings using clear visualizations

Source: NHANES 2020 Anthropometric Dataset

2️⃣ Exploratory Data Analysis (EDA)

Scatter plots for height vs weight

Gender-based comparative plots

4️⃣ Feature Engineering

Created derived anthropometric ratios

Compared proportional body measurement differences

📊 Key Insights

Strong positive correlation between height and weight across both genders.

BMI distributions show observable variability differences.

Waist circumference correlates significantly with weight.

Male and female distributions differ in spread and central tendency.
